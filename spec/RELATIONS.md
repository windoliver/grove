# Relation Types

> Work in progress. See issue #2.

## v1 Relations

| Type | Meaning |
|------|---------|
| `derives_from` | Built on this work's ideas or code |
| `responds_to` | Discussion reply |
| `reviews` | Evaluates quality or correctness |
| `reproduces` | Confirms or challenges results |
| `adopts` | Marks as valuable input (distinct from merge) |
